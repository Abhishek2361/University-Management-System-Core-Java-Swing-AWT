# University-Management-System-Core-Java-Swing-AWT
A desktop-based University Management System developed using Core Java, Swing, and AWT, designed to streamline academic and administrative operations within an educational institution.
This application provides an intuitive graphical user interface (GUI) to manage student records, faculty details, courses, and essential university activities efficiently.

Project Overview:
The University Management System is a Java desktop application that simulates real-world university workflows. It focuses on improving data organization, reducing manual effort, and enhancing usability through a structured GUI-driven design

Key Features:
✔ Student Information Management
✔ Faculty / Staff Management
✔ Course & Subject Handling
✔ Admission & Registration System
✔ Record Search & Updates
✔ User-Friendly GUI Interface
✔ Data Validation & Error Handling

Technologies Used:
Core Java
Java Swing
Java AWT
OOP Concepts
JDBC

Functional Modules:
Student Management – Add, update, delete, view student records
Faculty Management – Maintain staff details
Course Management – Organize courses & subjects
Administrative Operations – Centralized data handling

Objective:
The primary goal of this project is to demonstrate practical implementation of:

Object-Oriented Programming (OOP)
Java GUI Development
Event Handling
Application Structure & Design

Learning Outcomes:
Through this project, key software development concepts were explored:
✔ GUI Design Principles
✔ Java Event-Driven Programming
✔ Data Handling & Validation
✔ Modular Application Design
